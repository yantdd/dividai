# DividAí - O Seu Gestor de Despesas em Grupo 💰

O DividAí é uma aplicação web minimalista e direta para você nunca mais se perder nas divisões de contas com a galera. Seja no fim de semana na praia, no churrasco da turma, ou nos boletos eternos da república estudantil.

## 🎯 Público Alvo
- **Estudantes Universitários** compartilhando os mesmos espaços vivendo de república e as pequenas contas semanais.
- **Grupos de Viagem e Amigos**, que frequentemente lidam com aquela velha pergunta: *"quem pagou a pizza ontem?"*.

## ✨ Principais Funcionalidades
- **Gestão Fluida de Grupos**: Crie repúblicas, eventos ou "caixinhas" com dois toques.
- **Painel Financeiro Simplificado**: Cartões diretos avisando quem deve quanto ou de quem você tem a receber.
- **Mock Inteligente de Leitura de Nota Fiscal**: Demonstração de funcionalidade de Upload interagindo de forma simulada via IA pra extração do valor da conta e preenchimento de input ágil.
- **Cálculo Descomplicado**: Divisão flexível entre os membros ativados no grupo sem criar confusão.
- **Responsividade Híbrida Premium**: Feito a partir do princípio *Mobile First* permitindo visualizações gigantes em super telas de Desktop *Ultra-wide* sem espremer ou desalinhar nenhum formulário.

## 💻 Tech Stack
Construído com tecnologias rápidas, visuais e confiáveis em um setup React com:
- **React 18** empacotado e rodando agressivamente com o **Vite**
- **Tailwind CSS** para a composição inteira do sistema visual responsivo (utilitários CSS e modais dinâmicos).
- **React Router Dom (v6)** controlando a navegação interna ágil estilo *Single-Page Application*.
- **Context API** controlando todos os estados efêmeros via simulação em memória.
- **Lucide-React** alimentando toda a biblioteca consistente e viva de ícones.

## 🚀 Como Testar / Executar Localmente

### Pré-requisitos
Certifique-se de você tem o [Node.js](https://nodejs.org/) instalado.

1. **Clone ou Baixe esse repositório** para a sua máquina.
2. Acesse a raiz do repositório através de seu Terminal:
   ```bash
   # Navegar até a pasta local do repo
   # Ex: cd dividai
   ```
3. **Instale as dependências** via npm:
   ```bash
   npm install
   ```
4. **Acione o Servidor Front-end**:
   ```bash
   npm run dev
   ```
5. Pronto! É só acessar `http://localhost:5173` no seu navegador favorito e mergulhar em uma gestão transparente de conta com design Premium.
